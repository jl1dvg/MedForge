### ¿Qué hace?
El botón **Ver/Ocultar métricas** expande o colapsa tarjetas secundarias del Overview.

### ¿Cuándo usarlo?
- Cuando quieres una vista compacta para operación diaria.
- Cuando necesitas revisar métricas detalladas para seguimiento.

### Tip rápido
La preferencia queda guardada en este navegador para tu próxima sesión.
