### ¿Qué hace?
Abre el editor de datos quirúrgicos (LIO, poder, ojo, incisión, etc.).

### Label operativo
Editar datos quirúrgicos.

### Selector / ID
`#btnPrefacturaEditarLio`

### Ruta / Endpoint
`POST /solicitudes/{id}/cirugia`

### Qué significa para el flujo
Asegura que la planificación quirúrgica se mantenga actualizada antes de agenda.
