### ¿Qué hace?
Registra un bloqueo en agenda para coordinación del caso.

### Label operativo
Reservar espacio en agenda.

### Selector / ID
`#crmBloqueoForm`

### Ruta / Endpoint
`POST /solicitudes/{id}/crm/bloqueo`

### Qué significa para el flujo
Asegura disponibilidad de horario/sala para la atención planificada.
