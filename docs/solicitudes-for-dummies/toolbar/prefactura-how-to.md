### ¿Qué hace?
Explica cómo operar el modal de prefactura cuando la solicitud ya está abierta.

### Orden recomendado
1. Revisa **Resumen** y acciones rápidas.
2. Valida **Solicitud** y **Derivación**.
3. Confirma **Apto quirúrgico**.
4. Usa **Examen & Plan** para copiar texto clínico.
5. Cierra en **CRM** para seguimiento.

### Qué significa para el flujo
Estandariza el trabajo del equipo para que todas las solicitudes avancen con el mismo criterio.
