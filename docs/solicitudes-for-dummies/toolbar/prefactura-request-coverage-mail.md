### ¿Qué hace?
Abre el modal para redactar y enviar correo de cobertura.

### Label operativo
Redactar correo de cobertura.

### Selector / ID
`#btnPrefacturaSolicitarCoberturaMail`

### Ruta / Endpoint
UI local y envío posterior a endpoint de correo

### Qué significa para el flujo
Formaliza la solicitud de cobertura y deja registro de comunicación.
