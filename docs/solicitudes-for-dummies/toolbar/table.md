### ¿Qué hace?
Muestra las solicitudes en formato **tabla** para comparar datos fila por fila.

### ¿Cuándo usarlo?
- Cuando necesitas revisar muchos campos a la vez.
- Cuando quieres ordenar visualmente por columnas (estado, responsable, turno).

### Tip rápido
Usa el botón de ojo para abrir detalle y el botón CRM para seguimiento.
