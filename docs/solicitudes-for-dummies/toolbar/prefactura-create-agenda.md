### ¿Qué hace?
Abre el flujo para crear agenda quirúrgica contextual.

### Label operativo
Generar agenda quirúrgica.

### Selector / ID
`[data-context-action="generar-agenda"]`

### Ruta / Endpoint
`${basePath}/agenda?...`

### Qué significa para el flujo
Permite convertir una solicitud lista en una cita quirúrgica concreta.
