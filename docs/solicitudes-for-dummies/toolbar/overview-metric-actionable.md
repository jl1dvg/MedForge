### ¿Qué hace?
Cada tarjeta de métrica con ícono de reporte lanza un **quick report** filtrado por esa métrica.

### ¿Cuándo usarlo?
- Cuando necesitas un corte rápido para coordinación o auditoría.
- Cuando quieres exportar solo un segmento crítico (por ejemplo, SLA vencido).

### Tip rápido
Primero aplica filtros globales (fecha, doctor, afiliación) y luego exporta desde la tarjeta.
