### ¿Qué es este tablero?
Cada columna representa una etapa del proceso de la solicitud.

### Cómo usarlo (operativo)
- Arrastra tarjetas entre columnas para actualizar la etapa.
- Marca checklist para dejar tareas en completado o pendiente.
- Abre CRM para registrar llamadas y seguimiento.
- Usa Turnero para llamar al paciente cuando corresponde.

### Qué significa para el flujo
- **Pasar a siguiente etapa:** la solicitud avanza en el proceso quirúrgico.
- **Checklist:** deja trazabilidad del avance real.
- **Abrir CRM:** registra gestión y próximos pasos.
- **Llamar turno:** activa atención del paciente.
- **Drag & Drop:** corrige o actualiza estado manualmente.

### Mapa visual de referencia
Solicitud creada -> Revisión -> Programación -> Cirugía -> Facturación
